*REM: Funciona igual que el em, con la diferencia que es relativo al valor de la fuente del elemento HTML, y no tiene en cuenta el valor heredado o del elemento que lo contiene.

*REM: Tiene de referencia el estilo que tiene la etiqueta root por lo general (HTML) o el root de el proyecto, y el font-size del HTML por defecto siempre es 16px.

*1 rem = 16px.

*Si queremos aplicar REM de una forma mas sencilla para no tener que hacer tantos calculos seteamos el HTML con un font-size de 62.5%.

*Dandole a el HTML un valor de 10px ya que 16px - 62.5% = 10px.
