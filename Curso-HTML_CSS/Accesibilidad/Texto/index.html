¿Que es Accesibilidad?

En términos generales, cuando se dice que un sitio es accesible, se hace referencia a que el contenido del sitio está disponible y su funcionalidad puede ser operada, literalmente, por cualquier persona. Es fácil suponer que todos los usuarios pueden ver y usar un teclado, un mouse o una pantalla táctil, y pueden interactuar con el contenido de una página de forma normal. Esto puede conducir a una experiencia que funciona bien para algunas personas, pero crea problemas que van desde simples molestias hasta problemas para otros.
